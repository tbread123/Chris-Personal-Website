# Chris Personal Website

Static personal resume and portfolio website for Chris Huang.

## Overview

This site presents Chris Huang's resume, education, experience, ChuanLong publication work, Memory Project photos, and contact information. It is built as a simple static website using HTML, CSS, and JavaScript, so it can be hosted directly with GitHub Pages.

## Project Structure

- `index.html` - main website content
- `styles.css` - responsive layout and visual styling
- `main.js` - small interaction scripts
- `assets/` - resume PDF, project PDFs, and photos

## Local Preview

Open `index.html` directly in a browser, or run a simple local server:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

## Deployment

The site is intended to be deployed with GitHub Pages from the `main` branch.
